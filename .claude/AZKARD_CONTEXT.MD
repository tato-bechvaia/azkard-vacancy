# Azkard Project Context

## What Azkard is
Azkard is a career platform for Georgia's professional and tech job market. We target mid-to-senior salaried roles: engineers, data scientists, product managers, designers, finance professionals, bank/fintech specialists. We compete adjacent to (not against) Awork.ge, which serves the service/hourly economy.

## Target users
- **Job seekers:** tech professionals, mid-senior career individuals, recent CS/engineering graduates from KIU, TSU, Ilia, Agricultural University
- **Employers:** Bank of Georgia, TBC Bank, local tech companies, startups, international remote-first companies hiring in Georgia

## Competitive positioning
- Awork.ge = service economy (restaurant, retail, security, sales agents)
- jobs.ge / hh.ge = broad but dated UX
- **Azkard = professional knowledge economy, modern UX, tech-forward**

## Brand identity
- Dark theme, sophisticated
- Purple primary (#6B46F0-ish — check tailwind.config.js for exact)
- Modern, editorial feel — closer to Linear or Vercel than to traditional job boards
- Georgian-first language, English fallback
- Typography must handle Georgian script well

## Tech stack
- React + Vite + Tailwind CSS
- React Router
- Supabase/PostgreSQL backend (do not touch)
- Express backend on Railway (do not touch)
- Deployed on Vercel

## What we do NOT do
- Do not mimic Awork.ge's visual style (light theme, dense cards, service-economy icons)
- Do not rename or rebrand
- Do not translate Georgian to English
- Do not add features outside scope
- Do not refactor backend or API
- Do not install new dependencies without explicit approval
